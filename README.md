# DesignPattern
本项目主要是一些关于设计模式的练习。

参考书籍：《大话设计模式》

目前已完成：

1. simple_factory：简单工厂模式实现的简单计算器
2. strategy：策略模式实现简单计算器
3. decorator：装饰者模式
4. proxy：代理模式
5. factory：工厂模式
6. prototype：原型模式(浅拷贝与深拷贝的应用)
7. template_method：模板方法模式
8. facade：外观模式
9. builder：建造者模式
10. observer：观察者模式
11. status：状态模式
12. adapter：适配器模式
13. memento：备忘录模式
14. composite：组合模式
15. singleton：单例模式
16. bridge：桥接模式
17. command：命令模式
